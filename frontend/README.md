# RPG Session Organizer Frontend

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Technologies

- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- DaisyUI
- Lucide Icons
